A responsive web application that fetches and displays free-to-play games using the Free-to-Play Games Database API. Users can browse games by category, view detailed information in a modal, and explore different genres through a dynamic dropdown filter.

Features:

Fetches real-time data from an external API (RapidAPI â€“ Free-to-Play Games Database)
Filter games by category (MMORPG, Shooter, Strategy, Action, etc.)
Dynamic rendering of game cards
Detailed game information displayed in a Bootstrap modal
Loading spinner animation while fetching data
Error handling for failed API requests
Fully responsive design using Bootstrap
Custom dark pastel purple UI theme

Technologies Used:

HTML5
CSS3
Bootstrap 5
JavaScript (ES6+)
Fetch API
RapidAPI Integration


How It Works:

On page load, the app fetches and displays all free-to-play games.
Users can filter by category using the dropdown menu.
Clicking on a game card opens a modal with detailed information.
Smooth scrolling navigation improves user experience.
